---
title: Installation
description: Install minions-prompts for TypeScript/Node.js or Python.
---

## TypeScript / Node.js

```bash
npm install minions-prompts minions-sdk
# or
pnpm add minions-prompts minions-sdk
```

## Python

```bash
pip install minions-prompts
```

The Python package installs `minions-sdk` as a dependency automatically.

## CLI

```bash
npm install -g minions-prompts-cli
```

After installation, the `prompts` command is available globally:

```bash
prompts --help
```

## Requirements

- **TypeScript/Node.js**: Node.js 18+ with ESM support
- **Python**: Python 3.10+
- **CLI**: Node.js 18+

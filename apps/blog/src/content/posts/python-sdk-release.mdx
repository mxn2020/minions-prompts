# Bringing Prompt Version Control to Python

The Python ecosystem dominates AI and ML development. If you're building prompt-powered applications with LangChain, LlamaIndex, or direct API calls, you need proper prompt management — and now you can do it in Python.

Today we're releasing the **Minions Prompts Python SDK**.

<Callout type="info">
The Python SDK mirrors the TypeScript SDK's design while staying idiomatic — using Pydantic for validation, dataclasses for configuration, and async/await for I/O operations.
</Callout>

## Installation

```bash
pip install minions-prompts
```

## Template Management

```python
from minions_prompts import PromptRenderer, PromptChain
from minions_prompts.types import PromptTemplate

# Define a prompt template
template = PromptTemplate(
    name="summarizer",
    content="Summarize the following {{format}}: {{text}}",
    variables=["format", "text"],
    tags=["summarization"]
)

# Render with variables
renderer = PromptRenderer()
rendered = renderer.render(template, {
    "format": "research paper",
    "text": "Long paper content..."
})

print(rendered)
```

## Version Chains in Python

Track prompt evolution with structured version chains.

```python
from minions_prompts import PromptChain

chain = PromptChain()

# Add the initial version
chain.add_version(
    template_id="summarizer",
    content="Summarize the following {{format}}: {{text}}",
    changelog="Initial version"
)

# Add an improved version
chain.add_version(
    template_id="summarizer",
    content="You are a precise summarizer. Summarize the following {{format}} in {{maxWords}} words: {{text}}",
    changelog="Added word limit and persona"
)

# Get version history
for version in chain.get_history("summarizer"):
    print(f"v{version.number}: {version.changelog}")
```

## Prompt Scoring

Evaluate and compare prompt versions programmatically.

```python
from minions_prompts import PromptScorer

scorer = PromptScorer()

# Score a single prompt output
score = scorer.score(
    output="This is a concise summary of the key findings...",
    criteria={
        "max_length": 200,
        "min_length": 50,
        "required_keywords": ["key findings"]
    }
)

print(f"Score: {score.value:.2f} ({score.passed}/{score.total} criteria met)")
```

## Diffing Prompts

Compare prompt versions to understand exactly what changed.

```python
from minions_prompts import PromptDiff

diff = PromptDiff()

result = diff.compare(
    old_content="Summarize: {{text}}",
    new_content="You are a summarizer. Summarize in {{maxWords}} words: {{text}}"
)

for change in result.changes:
    print(f"  {change.type}: {change.description}")
```

<Callout type="tip">
The Python SDK is fully compatible with the TypeScript SDK's serialization format. Templates created in TypeScript can be loaded and used in Python, and vice versa.
</Callout>

## Exporting to LangChain

One of the most requested features — export your versioned prompts directly to LangChain prompt templates.

```python
from minions_prompts import PromptExporter

exporter = PromptExporter()
langchain_template = exporter.to_langchain(template)

# Use directly with LangChain
from langchain.chains import LLMChain
chain = LLMChain(llm=my_llm, prompt=langchain_template)
```
